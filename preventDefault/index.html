En esta página, tenemos una galería de productos con imágenes y al hacer clic en un producto, se muestra la imagen ampliada en un área específica de la página. En este escenario, se utiliza get attribute para obtener la URL de la imagen del producto seleccionado y set attribute para mostrar esa imagen ampliada. Aquí el primer ejemplo:

En este ejemplo, cada imagen de producto tiene un atributo adicional data-imagen-ampliada que almacena la URL de la imagen ampliada correspondiente. Cuando se hace clic en un producto, el código JavaScript utiliza get attribute para obtener esta URL y luego utiliza set attribute para mostrar esa imagen ampliada en el área designada.
